# Hard Competitive Suite

Implement `solutions/h01/solution.mjs` through `solutions/h10/solution.mjs`.

Read `PROBLEMS.md` for all task statements. Each task has exactly one public sample in `samples/`.

Useful commands:

- `npm run sample -- h01`
- `npm run sample:all`
- `npm run run -- h01 < custom.in`
- `npm run judge -- h01`
- `npm run judge -- all`

The judge is a black box: during the agent run it reports only `PASS` or `FAIL`.
