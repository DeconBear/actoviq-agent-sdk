# Competitive Programming Stress Suite

Implement `solutions/p01/solution.mjs` through `solutions/p10/solution.mjs`.

Each problem has exactly one public sample.

Commands:

- `npm run sample -- p01`
- `npm run sample:all`
- `npm run run -- p01 < custom.in`
- `npm run judge -- p01`
- `npm run judge -- all`

The hidden judge returns only `PASS` or `FAIL`. It does not reveal failing tests.
