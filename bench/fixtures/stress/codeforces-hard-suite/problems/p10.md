# P10 - Grid Path With Turns

Inspired by a recent grid path DP problem.

Count paths from the upper-left cell to the lower-right cell moving only right or down, avoiding blocked cells. Count only paths with at most `k` direction changes. Return the answer modulo `1000000007`.

Input:

- `n m k`
- `n` grid rows using `.` and `#`

Output the count.
