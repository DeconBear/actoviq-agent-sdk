# P04 - Growing Tree Game

Inspired by a recent tree game.

You are given a rooted tree with root `1`. A position's Grundy number is the mex of the Grundy numbers of its children. If the root Grundy number is non-zero, the first player wins; otherwise the second player wins.

Input:

- `n`
- parents of vertices `2..n`

Output `First` or `Second`.
