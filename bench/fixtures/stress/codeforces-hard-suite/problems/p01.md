# P01 - Balanced Pancake Equation

Inspired by a recent number-theory cooking problem.

For each query, find the minimum number of operations needed to reach exactly `c` units when one operation of type A adds `a` units and one operation of type B adds `b` units.

Formally, find non-negative integers `x` and `y` such that:

```text
a*x + b*y = c
```

If several pairs exist, minimize `x + y`. If no pair exists, print `-1`.

Input:

- `t`
- `t` lines with `c a b`

Output one answer per query.
