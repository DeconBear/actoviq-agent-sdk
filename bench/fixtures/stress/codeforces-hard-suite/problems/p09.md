# P09 - MEX Hull on Tree

Inspired by a recent MEX-on-tree problem.

You are given a tree and a permutation of values `0..n-1` assigned to vertices. For each mex prefix `0..i`, consider the smallest connected subgraph containing all vertices with values from `0` through `i`. Output the number of vertices in that subgraph. Equivalently, count the vertices in the union of all tree paths required to connect those selected vertices.

Input:

- `n`
- `n-1` edges
- values of vertices `1..n`

Output `n` integers.
